# BackLight Landing Page

Static landing page for BackLight, intended for GitHub Pages deployment.

## Plan

- Publish the site from this directory's repository using GitHub Pages.
- Point the primary CTA at a GitHub Release asset once the signed and notarized DMG is available.
- Replace the direct download CTA with a Mac App Store link after App Store approval.

## Files

- `index.html`: Single-page marketing site.
- `styles.css`: Dark-mode landing page styling and hero animations.
